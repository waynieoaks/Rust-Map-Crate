# Rust-Map-Crate
## An Oxide plugin for Rust

Map Crate automatically marks hackable crates on the in-game map with a blue circular marker - allowing players to quickly see what monument the crate has been dropped at.

![Map Crate](https://github.com/waynieoaks/Rust-Map-Crate/blob/main/images/Map-Crate.png?raw=true)

##  Features
* Marks hackable crates with a blue map indicator
* No configuration, commands, or permissions required
* Markers are removed automatically when the crate despawns or is looted
* Fully server-side and lightweight
* Compatible with PvE and PvP servers

## Permissions
There are no permissions requirments for this plugin.

## Configuration
There is no configutation options for this plugin.
